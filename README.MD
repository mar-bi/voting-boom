# Voting Boom  
Full-stack voting application.

The user can create, store, and share his/her polls; vote in the available pools, and see the poll results visualized with a bar chart.

## **Voting Boom** is built with:
- on the **_front-end_**:
  1. React,
  2. React-Router,
  3. Material-UI,
  4. Sass,
  5. Axios,  
  6. D3


- on the **_back-end_**:
  1. Node,
  2. Express,
  3. Passport,
  4. jsonwebtoken,
  5. MongoDB,
  6. Mongoose,
  7. bcrypt-nodejs,
  8. validator
